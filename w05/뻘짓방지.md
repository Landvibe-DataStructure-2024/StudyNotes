# 뻘짓 방지
### 뻘짓이란?
- 다른 사람들은 생각도 못하는 부분에서 혼자 ‘뻘짓’ 하는 것


### 1. 
```cpp
if(a=b){ }
```
### 2. 
```cpp
int a,b;

while(a--){
}
// a는 몇일까?
// 입력 어케받음?
```
### 3.
```c++
int a,b;
cin >> b >> a;
// 문제랑 반대로 입력 받기
```

### 4.
```c++
int t,n;
cin >> t >> n;
// t : 테스트 케이스 횟수, n : 데이터 사이즈
while(n--){
    // ,,,,
}
```
```c++
int arr[30];
int a;
//...
a = arr[30];
// 잘못된 접근 !!!!!!!!
```

**... 이외에도 다양한 뻘짓이 있습니다.**
